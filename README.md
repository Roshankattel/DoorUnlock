# JS-enabled Door Unlock Mongoose OS firmware

This is the JS Door Unlock Mongoose OS app. It gets installed by default at
[Mongoose OS installation step](https://mongoose-os.com/docs/). It has
a lot of functionality enabled - cloud integrations, JavaScript engine, etc.

## Installation

1. Open a terminal and run the `mos` command.
2. Select the port and set the board to "esp8266".
3. Run `mos build`.
4. Run `mos flash`.
